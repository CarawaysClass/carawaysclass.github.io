Input and Output Lab

Description

In this lab, you will share (via Slack) a "non-typical" input or output device that you use. By non-typical, I mean it isn't a traditional keyboard or a traditional mouse for Input and it isn't a traditional monitor or a traditional speaker for Output.

Learning Outcomes

At the end of this lab, you will be able to:

  • Identify different types of Input and Output devices.
  • Explain how Input devices work.
  • Explain how Output devices work.
  • Describe a specific non-typical Input or Output device that you use.

Assignment Tasks

  1. First, be sure to read our 'Chapter 06 Input and Output' SIMbook.
  2. Select a non-typical input or output device that you currently use.
  3. Take a picture of the device.
  4. Post the picture of your device and then explain how you use it. Remember: be sure that it is a non-typical input or output device.

Submitting Your Work

  • Create your image and description post in our #input-and-output-lab Slack channel before the due date.
