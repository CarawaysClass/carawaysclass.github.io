System Unit Lab

Description

In this lab, you will take the information that you recently read about in our 'System Unit' chapter and will apply that to an actual piece of hardware that you use.

Learning Outcomes

At the end of this lab, you will be able to:

  • Identify the different components that make up a system unit
  • Explain what each component does
  • Describe the components found in a System Unit that you use

Assignment Tasks

  1. First, be sure to read our 'Chapter 05 The System Unit' SIMbook.
  2. Select a System Unit that you currently use.
  3. Download and save a copy of the attached System Unit Lab template document (available below).
  4. Fill out the System Unit Lab document by adding the specific information about your system unit.
  5. The document includes call-outs for the basic information that you should find for just about any type of system unit. Feel free to add any other specifications about your system unit that you would like to share, e.g., system board, expansion slots, etc.

Submitting Your Work

  1. Save your final document as a Word .docx file with a filename that follows the naming convention: SystemUnitLab-AB.docx where the last two letters (AB) are your initials.
  2. Upload the .docx file to this Dropbox by the designated Due Date.
