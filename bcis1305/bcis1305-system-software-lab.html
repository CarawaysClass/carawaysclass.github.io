System Software Lab

Description

In this lab, you will take the information that you recently read about in our ‘System Software’ chapter and will use that to research and describe an example of system software that you use today. From the operating system that keeps your device running to the utility application that makes sure your device performs in optimal fashion, System Software is a critical component of any computer.

Learning Objectives

At the end of this lab, you will be able to:

  • Identify the different types of system software.
  • Explain the basic functions, features, and categories of system software.
  • Describe the difference between System Software and Application Software.

Assignment Tasks

  1. First, read the ‘Chapter 03 Application Software’ and ‘Chapter 04 System Software’ SIMbook assignments.
  2. Next, identify the Operating System and one Utility that you currently use on a computing device of your choice. This device could be your laptop, your tablet, your phone, your gaming system, etc.
  3. Download and save a copy of the System Software Lab template document.
  4. Fill out the System Software Lab document by adding the specific information about your Operating System and your Utility Software.

Submitting Your Work

  1. Save your final document as a Word .docx file with a filename that follows the naming convention: SystemSofwareLab-AB.docx where the last two letters (AB) are your initials.
  2. Upload the .docx file to this Dropbox by the designated Due Date.
