Programming

Overview

Why should you need to know anything about programming (i.e., coding)? The answer is simple. You might need to deal with programmers one day in your work. Or, you may someday find yourself rolling up your sleeves to begin coding yourself. There's a growing trend toward end-user software development. This means that end users, like you, are developing their own application programs. Essentially, "scratching their own itch".

In fact, you've even done some coding in our class already when you were building formulas and other logic in your Excel spreadsheets!

Objectives

At the end of this week, you will be able to:

  • Define programming.
  • Identify types of programming languages and tools used in programming.
  • Write a simple program in Python.
