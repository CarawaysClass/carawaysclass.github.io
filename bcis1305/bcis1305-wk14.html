Week 14 - Integrated Project

Overview

This week you begin working on your Integrated Project in SIMnet - this is similar to the previous SIMnet projects that you have completed except this one includes all three Office applications: Word, PowerPoint and Excel. You have two weeks to complete this assignment.

We will also have a short mid-week assignment introducing the topic of Data Analytics.

Objectives

At the end of this week, you will be able to:

  • Use Microsoft Word, PowerPoint, Excel, and a database in a single project.
  • Explain what Data Analytics is in relation to Business Intelligence and Big Data

Learning Activities

  1. Review notes and pertinent references.
  2. Complete Data Analytics assignment(s) - these will be assigned later in the week.

Assignments

  2. Complete Data Analytics assignment(s) - watch for this to be available by Wednesday of this week.
