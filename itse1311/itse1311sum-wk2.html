Week 2 - HTML Markup for Structure

Overview

Read chapters 5, 6, 7, and 8 of our textbook. Be sure to follow along with author's example of creating a simple HTML file and note the resource links that are included in the text for starting files and other resources used in the exercises. After reading, complete the Week 2 Quiz.

Objectives

At the end of this week, you will be able to:

  • Use elements to structure a web page.
  • Describe div and span elements.
  • Use special characters.
  • Create links to external pages, relative links to internal documents, and other special applications.
  • Add images to a web page.
  • Use the src, alt, width, and height attributes.
  • Describe how tables are used.
  • Construct basic tables.
  • Set table headers and rows.
  • Make tables accessible.

Learning Activities

  1. Read Chapter 5: Marking Up Text.
  2. Read Chapter 6: Adding Links.
  3. Read Chapter 7: Adding Images.
  4. Read Chapter 8: Table Markup.
