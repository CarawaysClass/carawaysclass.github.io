Simple HTML Programming Challenge

In this programming challenge, we will introduce the basic concepts and skills required to create and deploy a simple HTML page to our web server.

In completing this assignment, the student will:

  • have direct experience using a text editor to create an html page
  • understand how textual and image content is added to an html page
  • have a basic understanding of deploying their html page to a server-based hosting service

Deliverables

In this programming challenge you will use a text editor to create a single HTML document that represents your personal resume page.  This page should include the following content items:

  1. a profile photo of you  
  2. a brief biographic description
  3. a list of your education and/or work experience
  4. a hyperlink to Lone Star College - this could be referenced in the education section of the document
  5. headings to identify the title and and subheadings for each section of this document

Submit Your Work

Name your file as resume.html and upload this to your SiteGround hosting service. Submit your work by sharing the URL to your page in the #first-html Slack channel.
