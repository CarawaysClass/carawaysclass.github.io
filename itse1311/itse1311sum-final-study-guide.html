ITSE 1311 - Final Exam Study Guide

This Study Guide is offered to help you prepare for our Final Exam. Our Final Exam will consist of:

  • 25 “Multiple Choice/True-False” questions (4 pts each)
  • 1 Short “Essay” question (25 pts)

The 25 “Multiple Choice/True-False” will be derived from the following topics/questions. In most cases, the actual exam question will not be worded exactly as these topics/questions.

  1. Describe what W3C stands for.
  2. What is the role of the W3C?
  3. Explain why we should use an alt tag in our image elements.
  4. What elements should every html document have?
  5. How do we create a horizontal line in html?
  6. What are examples of void elements?
  7. How do we reference the location of the image file inside of an img element?
  8. How do we reference the location of the linked file inside of an a element?
  9. Should headings (H1, H2, etc.) be used specifically for their styling appearance? Why or why not?
  10. Compare/contrast graceful degradation and progressive enhancement.
  11. What are some ways to improve the performance (load time, responsiveness, etc.) of your site?
  12. Why should we consider using a text editor over a WYSIWYG editor when creating our html/css?
  13. Explain how to comment code in an html file.
  14. Explain how to comment code in a css file.
  15. What is a semantic element in html?
  16. What is the purpose of the DOCTYPE?
  17. Using HTML, how do we change the title displayed in a browser tab or window?
  18. What does the body element contain?
  19. What element is used to define a paragraph in html?
  20. What's a simple way to force a new line in an html document?
  21. Why would we use an iframe (or in-frame) element?
  22. What are the html elements that are used in defining lists of content?
  23. How do we set the transparency for a background color?
  24. Know the different types of form input elements.
  25. Explain what makes up a font stack and how it works.
  26. How do we incorporate Google fonts in our own web pages?
  27. What are the differences between an embedded and external style sheet?
  28. Explain the difference between the margin and padding of an element.
  29. Know how to set a border for an element.
  30. What is the value in creating a site map?
  31. What tools can be used to create a wireframe?
  32. How does setting a float work?
  33. What makes an element a block-level element?

Essay Questions

The short “Essay” questions will be taken from the following pool of three questions. The actual exam questions will be worded exactly as the questions you find here.

  1. Explain the similarities and the difference between an id attribute and a class attribute as used in HTML and CSS. Provide an example of each in your explanation.
  2. Describe the CSS Box model and its different components. Provide an example in your explanation.
  3. Explain the similarities and differences of the following types of transitions: ease, ease-in, ease-out, and ease-in-out.
