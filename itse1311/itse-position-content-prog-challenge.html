Positioning Content Programming Challenge

Objectives

Upon completing this assignment, the student should…

  • Understand how an external stylesheet (CSS) is “linked” and used in conjunction with an .html content file.
  • Be able to position page content using CSS.
  • Further understand the effect of the box model on content width.
  • Be able to identify difference between relative and absolute positioning.
  • Be able to differentiate between pixel width and percentage width.
  • Have a basic understanding floats.

Time to complete: 60 min

Introduction

Using a partially completed .html and .css file set (derived from the the Jenware example of chapters 14 and 15), you will make the necessary corrections to the markup (html) and add the necessary styling (css) to make your page match a demonstrated final version of the tshirts4all.com products page similar to the Jenware.com products page.

Tasks

  1. Download and unzip the starter tshirts4all.zip file (included below). This includes the .html, .css and other resources required to complete this assignment.
  2. View this short screencast demonstrating the completed page and its behavior.
  3. Using your text editor, open the newproducts.html and style.com documents.
  4. Update the markup (html) to properly reference the image files - these are currently displayed as broken images when the newproducts.html page is opened in a browser.
  5. Update the “Site hand-crafted…” link at the bottom of the page to reference you and your SiteGround-hosted site.
  6. Following the CSS-specific Exercise steps 14–2, 14–3, 15–1 thru 15–5 (in our text), apply the necessary styling to make your page match what was demonstrated in step 2. Note that style.css is a complete (and long) stylesheet file - all of your edits should come below line number 490 (i.e., the content above line 490 should remain as-is and intact).
  7. When you are done, zip up your completed solution including your .html file, your .css file, and any resource folder(s) and upload the resulting .zip file into the corresponding dropbox found in D2L.
